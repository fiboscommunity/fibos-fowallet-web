FO 钱包官网

## 项目说明

本项目基于 vue 开发的前端项目，在使用之前请保证已安装好 `npm` 和 `node` 。

## 配置

配置路径: `src/util/config.js`

| Param          | Effect                                                         |
| --------------- | ------------------------------------------------------------ |
| hostUri  | 钱包部署地址                                                     |

## 安装及开发

`npm install`

`npm run dev`

## 打包

`npm run build`

打包后的文件存放在 `build` 文件夹中
